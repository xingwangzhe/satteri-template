satteri-mermaid 示例

plugin Satteri MDAST 插件：检测 Markdown 中的 mermaid 代码块， 将其转为 <pre class="mermaid">，配合 mermaid 库在浏览器端渲染 SVG。

静态示例（模拟插件输出）

flowchart LR

  B -->|mermaid.js| C[SVG 图表]
    
sequenceDiagram
  participant U as 用户
  participant S as Satteri
  participant M as Mermaid
  U->>S: 写 ```mermaid 代码
  S->>S: mermaidPlugin 检测并转换
  S->>M: 浏览器中调用 mermaid.run()
  M->>U: 渲染为 SVG
